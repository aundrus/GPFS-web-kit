SCDF GPFS Web Server Migration

Modernizing the SCDF GPFS dashboard web service from RHEL 6 / PHP 5.3 to RHEL 9 / PHP 8.2, with a clean HTML5 front-end for a safer, more maintainable platform.

Status: Ongoing (since Feb 17) Target: RH9 + PHP 8.2

Purpose

What we’re doing
  • Move off end-of-life infrastructure and align with supported security baselines.
  • Keep the dashboard familiar while improving readability and responsiveness.
  • Reduce operational risk and simplify long-term maintenance.

Advantages

Why it’s worth it
  • Security: supported OS/runtime, modern TLS/browser compatibility.
  • Reliability: fewer legacy dependencies, cleaner runtime behavior.
  • Maintainability: easier updates, clearer code paths, better tooling.
  • User experience: responsive HTML5 layout that looks good on modern screens.
